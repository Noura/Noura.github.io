A simple interface enables creating logic commands between input pins and output pins on the microcontroller. For example, if analog sensor input pin A0 reaches a threshold value, turn on whatever is connected to digital output pin 8.

The Intel Labs project ConnectAnyThing (release repo, development repo) makes it possible to use Intel's Galileo microcontroller without writing a line of code. It enables creative play and rapid prototyping. I worked on the UI design as well as client and server side code, for real-time responsiveness and consistent state across the microcontroller, multiple clients interacting simultaneously, and the server.

MakerNode (repo), a work in progress, is intended to provide a simple workflow for node.js projects on Galileo.
